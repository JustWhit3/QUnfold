Detailed Description

This class represents a binary encoder for arrays.

Encoding is based on a non-standard n-bit integer encoding: since current quantum computing hardware requires the bit encodings to be small each array value x_i is encoded with n bits using an offset alpha_i, and a scaling parameter, beta_i.
